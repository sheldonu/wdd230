Grid Layouts

churchofjesuschrist.org

The website for the Church uses Grid Layouts exceptionally well. They make everything look perfectly spaced out, so that the site is more clean and easily accessible.

Web Safe Fonts

Furniture Plus

furnitureplus.com

This website is a great example of using the principle of Web Safe Fonts. Throughout the website it uses the fonts: Arial and sans-serif, each of which are almost always able to display no matter what device is used.
